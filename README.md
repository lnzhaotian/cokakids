# 可卡英语在线课堂操作说明
## 学员
### 移动端进入教室（苹果设备为例）
1. 在微信中点击老师发来的教室链接（图1）；
2. 打开页面后点击右上角的“...”（图2）；
3. 在下面弹出的菜单中点击“在Safari中打开”（安卓设备中为“在浏览器打开”）（图3）；
4. 跳转到浏览器中页面加载完成后，如果已有帐户则点击“登录”，如果是首次使用则点击“注册”（图4）；
5. 输入自己帐户的邮箱地址和密码并点击登录（图5.1），或填写表单注册新帐户后登录（图5.2）；
6. 回到教室页面点击“加入”按钮（图6）；
7. 老师尚未进入教室之前会显示等待画面，老师进入教室之后学员进入到教室内（图7）
